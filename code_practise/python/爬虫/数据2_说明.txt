【数据说明 2：Bilibili 视频互动特征数据集】

数据背景与概况
本数据集采集自 Bilibili（B站），这是国内最大的年轻人视频社区。
为了保证样本的多样性，我们没有采用单一的热门榜单，而是通过 API 接口对全站 15 个主流频道（包括科技、游戏、知识、生活、动画等）进行了分层采样。
本次采集共获得 2122 条有效视频数据，用于分析用户互动行为对视频点赞率的影响。

字段定义
视频标题：视频的文字描述，作为样本的唯一标识。
所属分区：视频所在的业务板块（如科技区、生活区）。在回归分析中可作为调节变量。
播放量：视频的总观看次数。作为核心自变量（X1），衡量视频的基础曝光度。
弹幕数：用户在观看过程中的实时互动量。作为控制变量（X2），反映视频的社区氛围。
收藏数：用户将视频存入收藏夹的次数。作为特征变量（X3），反映内容的长期价值。
投币数：用户对高质量内容的物质认可。作为质量变量（X4），反映内容的硬核程度。
点赞数：用户对视频最直接的正向表态。作为目标因变量（Y），用于衡量视频的整体满意度。

采集与预处理技术
采集过程通过调用 B站 开放 API 实现，避免了网页端反爬机制对数据的干扰。
在清洗阶段，我们针对跨分区出现的重复视频进行了全局去重。
所有计数类字段（如点赞、播放）均从原始字符串格式转换为数值型（Integer/Float），以便于计算相关系数及拟合回归方程。
