【数据说明 1：晋江文学城作品数据集】

数据背景与概况
本数据集采集自晋江文学城（jjwxc.net），
该平台是中国最具影响力的女性向网络文学创作基地。数据主要涵盖了该平台全本排行库中的头部及中部作品。
本次采集共计获得 2700 条独立记录，旨在研究网络文学作品的体量（字数）与市场热度（积分）之间的量化关系。

字段定义
作者：作品的创作者姓名。用于识别样本来源。
书名：作品的唯一标题。
类型：作品的风格标签（例如：原创-纯爱-架空历史）。在统计中作为分类特征，用于观察不同题材的市场表现差异。
字数：作品完结时的正文总字数。在回归模型中作为核心自变量（X）。
作品积分：平台根据点击、收藏、评论等维度加权计算出的综合评分。在模型中作为因变量（Y），代表作品的商业与人气价值。
发表时间：作品首次发布的时间戳。

采集与预处理技术
数据通过 Python 编写的自动化脚本采集。
在预处理阶段，我们对“字数”进行了标准化处理，将文本单位（如“万”）转换为纯数值格式。
同时，基于“书名+作者”的唯一性原则进行了去重，并剔除了积分异常或字数过少的样本，确保了 2000 条以上数据的建模有效性。
